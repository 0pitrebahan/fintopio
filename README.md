## Persiapan

## Start Bot
```
sudo apt-get install git
```
```
git clone https://github.com/0pitrebahan/fintopio.git
cd fintopio
```
```
npm install
```
Edit `data.txt` (Kalian submit query id telemu)
```
nano data.txt
```

```
node main.js
```
